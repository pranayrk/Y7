### 1.5 Digital Text and Numbers ##### How do computers recognize text and numbers? * Any computer can understand only binary numbers * This means that even the letters of the alphabet have to be a binary number to make sense to the computer * **ASCII** is a widely used system to convert the alphabet (and numerals) to binary numbers.
### 1.5 Digital Text and Numbers ##### What is ASCII? * **ASCII** (American Standard Code for Information Interchange) is a system that computers use to represent text. * Each letter, number, and symbol on your keyboard has its own special number in the ASCII code.
### 1.5 Digital Text and Numbers * For example: * The letter 'A' is represented by the number 65. * The letter 'a' is represented by the number 97. * The number '1' is represented by the number 49. * The **standard ASCII** system includes 128 codes, ranging from 0 to 127. * There is also an **extended ASCII** system that includes 256 codes. * These codes include letters, numbers, punctuation marks, and control characters (like newline)
### 1.5 Digital Text and Numbers * Each ASCII character is represented in binary (base 2) inside the computer. For example, the letter 'A' (65 in decimal) is 01000001 in binary. * The problem with ASCII is that there are only 256 characters * This is because ASCII was originally created to deal with English * An improved code called **Unicode** was invented in 1991 to deal with other languages Homework: What is different in how ASCII and Unicode store characters?
