Task: Sum of natural numbers

Calculate the sum of natural numbers using a while loop.

	
n = int(input("Enter a number: "))
i = 1
total = 0
while i <= n:
    total += i
    i += 1
print("The sum is: ", total)
Task: Multiplication Table

Generate a multiplication table for a given number using a while loop.

	
num = int(input("Enter a number: "))
i = 1
while i <= 10:
	print(num, " x ", i , " = ", num * i)
	i += 1
